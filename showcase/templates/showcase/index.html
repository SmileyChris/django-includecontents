Quick Start

Get up and running with Django IncludeContents in minutes

Use HTML-like syntax for components

<include:forms:button label="Click me!" variant="primary" size="lg" />

<include:layout:card title="Card Title">
    <p>Card content goes here</p>
</include:layout:card>

<include:feedback:alert type="success">
    Operation completed successfully!
</include:feedback:alert>

Traditional Django template syntax

{% templatetag openblock %} load includecontents {% templatetag closeblock %}

{% templatetag openblock %} includecontents "components/forms/button.html" with label="Click me!" variant="primary" size="lg" {% templatetag closeblock %}
{% templatetag openblock %} endincludecontents {% templatetag closeblock %}

{% templatetag openblock %} includecontents "components/layout/card.html" with title="Card Title" {% templatetag closeblock %}
    <p>Card content goes here</p>
{% templatetag openblock %} endincludecontents {% templatetag closeblock %}

{% templatetag openblock %} includecontents "components/feedback/alert.html" with type="success" {% templatetag closeblock %}
    Operation completed successfully!
{% templatetag openblock %} endincludecontents {% templatetag closeblock %}

Installation and configuration

# Install the package
pip install django-includecontents

# Add to INSTALLED_APPS
INSTALLED_APPS = [
    ...
    'includecontents',
]

# Optional: Use HTML syntax
TEMPLATES = [{
    ...
    'BACKEND': 'includecontents.django.DjangoTemplates',
}]
