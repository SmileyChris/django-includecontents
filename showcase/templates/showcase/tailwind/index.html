üé® Tailwind 4.0 @theme CSS

Create CSS files with @theme directives:

@theme {
  --color-primary: #3b82f6;
  --spacing-lg: 2rem;
  --font-family-heading: 'Inter', sans-serif;
}

Usage: var(--color-primary) or class="text-primary"

‚öôÔ∏è tailwind.config.js

Add a Tailwind configuration file:

// tailwind.config.js
module.exports = {
  theme: {
    colors: {
      primary: '#3b82f6',
      secondary: '#64748b'
    },
    spacing: {
      'lg': '2rem'
    }
  }
}

Usage: class="text-primary bg-secondary p-lg"

Tokens will appear automatically when you refresh the page after adding any of the above files.

For comprehensive discovery, run: python manage.py import_tailwind_tokens
